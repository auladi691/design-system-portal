# Asset System

## Types

Icons, Icon Illustrations, Illustrations, Logos, Brand Assets, Templates, Downloads.

- Icons use Outline style only; do not add a style filter.
- Icon Illustrations are expressive small visuals and require brand metadata: Shared, IM3, Indosat, Tri, or Partner.
- Illustrations are larger storytelling scenes.

## Asset metadata

Name, slug, description, type, category, keywords, aliases, brand, status, version, preview, downloadable files, Figma URL, theme compatibility, accessibility guidance, related assets.

SVG icons should use `currentColor`. Assets may define separate light/dark files. Show usage references before archive/delete.
